================================================================================
                                  NOTES
================================================================================

1. Components should be flexible - sizes can adapt based on content
2. Use semantic colors for meaningful UI elements
3. Maintain 16px base grid for spacing alignment
4. All interactive elements need hover/active states
5. Dark theme uses #0A0A0A or #1A1A1A backgrounds
6. Light theme uses #F6F6F6 or #FFFFFF backgrounds
7. Green (#14E25A) is the primary accent - use consistently
8. Avoid mixing #14E25A with #00E676 - stick to #14E25A
